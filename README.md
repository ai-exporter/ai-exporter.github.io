# Chat Exporter for Google AI Studio

A free Chrome extension to export your Google AI Studio conversations as beautifully formatted PDF, Markdown, or JSON files.

## Features

- **One-Click Export**: Export any chat with a single click.
- **Multiple Formats**:
  - **PDF**: Clean, professional layout with timestamps and model names.
  - **Markdown**: Easy-to-read format perfect for notes and documentation.
  - **JSON**: Structured data for developers and automation.
- **Privacy-First**: All processing happens locally in your browser. No data is ever sent to any server.
- **No Account Required**: Use it instantly without signing up or logging in.
- **Dark Mode Support**: Exports maintain your theme preference.
- **Progress Tracking**: Real-time progress bar during export.

## How It Works

1. **Install the Extension**: Add "Chat Exporter for Google AI Studio" from the Chrome Web Store.
2. **Open AI Studio**: Navigate to any chat conversation.
3. **Click the Icon**: Click the extension icon in your browser toolbar.
4. **Choose Format**: Select PDF, Markdown, or JSON.
5. **Download**: The file is generated and downloaded instantly.

## Installation

### Chrome

1. Open the [Chrome Web Store](https://chromewebstore.google.com/).
2. Search for "Chat Exporter for Google AI Studio".
3. Click "Add to Chrome" and confirm the installation.

## Usage

1. Go to [Google AI Studio](https://aistudio.google.com/).
2. Open any chat conversation.
3. Click the extension icon in the top-right corner of your browser.
4. Select your desired export format.
5. The file will be downloaded automatically.

## Privacy Policy

We take your privacy seriously. This extension processes all chat data locally in your browser. No conversations are ever stored on our servers or transmitted to any third party. For more details, please read our [Privacy Policy](https://rajat-xr.github.io/privacy).
